# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project follows the version declared in `metadata.json`.

## [Unreleased]

### Added

- GNOME Shell 50 compatibility.
- Open source project documentation and GitHub collaboration templates.

## [1.0.0] - 2026-07-21

### Added

- Hebrew date display in the GNOME top panel.
- Hebrew date display in the clock/calendar menu.
- Preferences for selecting a location.
- Shkiah-aware date updates when a location is configured.
- Bundled GJS-compatible zmanim and Hebrew calendar calculations.
- Repository checks and GNOME Shell extension packaging script.

### Changed

- Repository organized to separate extension entrypoints, internal modules, documentation, assets, and scripts while preserving GNOME Shell compatibility.
